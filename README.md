# Painel de Bordo LEAD

Projeto React/Vite pronto para publicar na Vercel.

## Rodar localmente

```bash
npm install
npm run dev
```

## Publicar

Suba esta pasta para um repositório GitHub e conecte na Vercel.

## Google Apps Script

Quando a URL do Google Apps Script estiver pronta, abra `src/App.tsx` e preencha:

```ts
const GOOGLE_SCRIPT_URL = "SUA_URL_AQUI";
```
